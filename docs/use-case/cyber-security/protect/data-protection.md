Data protection focuses on reducing the risk and impact of data loss,
theft, or misuse while enabling the business to use data safely.

## Outcomes

- Reduced likelihood and impact of data breaches
- Improved compliance with privacy and sector regulations
- Clear handling rules for sensitive data and regulated records
- Better detection and containment of data exfiltration attempts

## Typical scope

- Data handling rules by classification and criticality
- Encryption and key management expectations
- Data loss prevention (DLP) for key channels
- Monitoring and response for high-risk data movement
